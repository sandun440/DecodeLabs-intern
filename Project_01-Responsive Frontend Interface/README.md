# Project 01 — Responsive Frontend Interface

A modern, responsive landing page built with semantic HTML, CSS grid/flexbox layout, and lightweight JavaScript.

## Project Overview

This interface demonstrates a polished marketing-style website with:
- sticky top navigation and mobile hamburger menu
- hero section with call-to-action buttons
- services, about, and portfolio sections
- contact footer and sign-up modal
- responsive layout for large screens and mobile

## Files

- `index.html` — project markup and main page structure
- `style.css` — responsive styles, layout, typography, and visual polish
- `script.js` — mobile menu toggle, modal interactions, smooth scrolling, and form behavior

## How to Run

1. Open `index.html` in your browser.
2. Or use a local server if preferred (e.g. VS Code Live Server extension).

## Key Features

- responsive hero layout with split visual/content sections
- mobile-friendly nav drawer and animated hamburger icon
- reusable section cards for services and portfolio
- accessible smooth scrolling navigation
- modal sign-up form with client-side submission feedback
