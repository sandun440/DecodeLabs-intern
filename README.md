# DecodeLabs Intern Workspace

This repository contains internship practice projects for DecodeLabs.

## Projects

- `Project_01-Responsive Frontend Interface/`
  - A responsive frontend interface project with HTML, CSS, and JavaScript.
  - Includes `index.html`, `style.css`, and `script.js`.

- `Project_02- Backend API Developmert/`
  - A Node.js backend API built with Express.
  - Includes `index.js`, `package.json`, and a project-level `README.md`.

-- `Project_03- Database Integration/`
  - PHP + MySQL CRUD example demonstrating database connection and basic Create/Read/Update/Delete operations.
  - Files include `index.php`, `db.php`, `insert.php`, `view.php`, `edit.php`, `update.php`, `delete.php`, and `style.css`.
  - See the detailed project README: `Project_03- Database Integration/README.md` for setup, SQL to create the `students` table, and run instructions.

- `Project_04-/`
  - Reserved for future development or practice tasks.

## How to use

- Open the project folder you want to work on in your editor.
- For frontend work, open `Project_01-Responsive Frontend Interface/index.html` in a browser.
- For backend work, install dependencies in `Project_02- Backend API Developmert/` and run the server.
 - For PHP + MySQL (Project_03): follow the SQL and setup in `Project_03- Database Integration/README.md`, update credentials in `Project_03- Database Integration/db.php` if needed, then run the project with a local web stack (XAMPP/WAMP/MAMP) or the PHP built-in server:

```bash
cd "Project_03- Database Integration"
php -S localhost:8000
# Open http://localhost:8000/index.php
```
