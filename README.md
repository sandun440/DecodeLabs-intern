# DecodeLabs Intern Workspace

This repository contains internship practice projects for DecodeLabs.

## Projects

- `Project_01-Responsive Frontend Interface/`
  - A responsive frontend interface project with HTML, CSS, and JavaScript.
  - Includes `index.html`, `style.css`, and `script.js`.

- `Project_02- Backend API Developmert/`
  - A Node.js backend API built with Express.
  - Includes `index.js`, `package.json`, and a project-level `README.md`.

- `Project_03-/`
  - Reserved for future development or practice tasks.

- `Project_04-/`
  - Reserved for future development or practice tasks.

## How to use

- Open the project folder you want to work on in your editor.
- For frontend work, open `Project_01-Responsive Frontend Interface/index.html` in a browser.
- For backend work, install dependencies in `Project_02- Backend API Developmert/` and run the server.

## Notes
